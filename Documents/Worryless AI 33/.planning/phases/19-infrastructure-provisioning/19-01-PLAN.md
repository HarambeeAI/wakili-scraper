    The executor cannot run arbitrary SQL via MCP tools. Present the user with the DATABASE_PUBLIC_URL from Task 1 and ask them to connect via psql and run these SQL commands:

    1. Connect: `psql "<DATABASE_PUBLIC_URL>"`
    2. Enable pgvector: `CREATE EXTENSION IF NOT EXISTS vector;`
    3. Create logto database: `CREATE DATABASE logto;`
    4. Verify pgvector: `SELECT extname, extversion FROM pg_extension WHERE extname = 'vector';` (expect one row)
    5. Verify logto db: `SELECT datname FROM pg_database WHERE datname = 'logto';` (expect one row)
    6. Exit: `\q`
