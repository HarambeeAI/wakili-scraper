**Auth.tsx — Complete rewrite per D-13, D-16:**

Replace the entire file. Remove ALL Supabase imports. The new Auth.tsx:
1. Imports: `useAuth` from `@/hooks/useAuth`, `useLogto` from `@logto/react`, `useNavigate` from `react-router-dom`, `useEffect` from `react`
2. If `isAuthenticated` is true, `navigate("/dashboard")`. Otherwise, call `signIn()` which redirects to Logto hosted login UI.
3. Renders a minimal "Redirecting to sign in..." loading state.
4. NO custom sign-in/sign-up form — Logto provides its own hosted UI.

Exact implementation:
```typescript
import { useEffect } from "react";
import { useNavigate } from "react-router-dom";
import { useLogto } from "@logto/react";
import { useAuth } from "@/hooks/useAuth";

const Auth = () => {
  const { signIn } = useAuth();
  const { isAuthenticated } = useLogto();
  const navigate = useNavigate();

  useEffect(() => {
    if (isAuthenticated) {
      navigate("/dashboard");
    } else {
      signIn();
    }
  }, [isAuthenticated, signIn, navigate]);

  return (
    <div className="min-h-screen flex items-center justify-center bg-background">
      <p className="text-muted-foreground">Redirecting to sign in...</p>
    </div>
  );
};

export default Auth;
```
