**Create worrylesssuperagent/Dockerfile per D-22:**

Multi-stage build. Stage 1 builds Vite app with Node.js, Stage 2 serves with Nginx.

```dockerfile
# Stage 1: Build the Vite application
FROM node:20-alpine AS builder
WORKDIR /app
COPY package*.json ./
COPY bun.lock* ./
RUN npm ci
COPY . .

# VITE_ vars must be passed as build arguments (baked into JS bundle at build time)
ARG VITE_API_URL
ARG VITE_LOGTO_ENDPOINT
ARG VITE_LOGTO_APP_ID
ARG VITE_LOGTO_API_RESOURCE
ARG VITE_VAPID_PUBLIC_KEY
ARG VITE_LANGGRAPH_URL

RUN npm run build

# Stage 2: Serve with Nginx
FROM nginx:1.27-alpine
COPY --from=builder /app/dist /usr/share/nginx/html
COPY nginx.conf /etc/nginx/conf.d/default.conf
EXPOSE 80
CMD ["nginx", "-g", "daemon off;"]
```

CRITICAL: All `VITE_*` env vars are declared as `ARG` in the Dockerfile because Vite substitutes them at build time. Railway must pass these as build arguments — they cannot be set as runtime environment variables. Railway automatically makes service variables available as build args when `ARG` declarations match the variable names.

**Create worrylesssuperagent/nginx.conf per D-23:**

```nginx
server {
    listen 80;
    root /usr/share/nginx/html;
    index index.html;

    # Gzip compression
    gzip on;
    gzip_types text/plain text/css application/javascript application/json image/svg+xml;
    gzip_min_length 1024;
    gzip_vary on;

    # SPA fallback — all non-file routes serve index.html
    location / {
        try_files $uri $uri/ /index.html;
    }

    # Cache static assets with content-hashed filenames (immutable)
    location /assets/ {
        expires 1y;
        add_header Cache-Control "public, immutable";
    }

    # Cache other static files (images, fonts, etc.)
    location ~* \.(js|css|png|jpg|jpeg|gif|ico|svg|woff2?)$ {
        expires 30d;
        add_header Cache-Control "public";
    }

    # Never cache the HTML shell (SPA entry point must always be fresh)
    location = /index.html {
        add_header Cache-Control "no-cache, no-store, must-revalidate";
        add_header Pragma "no-cache";
        add_header Expires 0;
    }
}
```
