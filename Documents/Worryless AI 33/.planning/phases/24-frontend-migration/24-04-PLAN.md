Supabase call replacement map:
- `supabase.from("table").select("*").eq("user_id", userId)` -> `api.get<T[]>("/api/table", { token })`
- `supabase.from("table").insert({ ... })` -> `api.post<T>("/api/table", body, { token })`
- `supabase.from("table").update({ ... }).eq("id", id)` -> `api.patch<T>("/api/table/${id}", body, { token })`
- `supabase.from("table").delete().eq("id", id)` -> `api.delete("/api/table/${id}", { token })`
- `supabase.functions.invoke("fn-name", { body })` -> `api.post("/api/fn-name", body, { token })`
