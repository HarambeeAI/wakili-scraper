# Phase 16 — UI Design Contract

> Visual and interaction contract for Phase 16: Proactive Cadence Engine.
> Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Scope Note

Phase 16 is primarily a backend and Edge Function phase. The only frontend surface is an
extension to the existing `HeartbeatConfigSection` component inside `GenericAgentPanel`.
The new `CadenceConfigSection` adds multi-tier cadence toggle controls (daily / weekly /
monthly / quarterly) and an event-triggers toggle that map to the `cadence_config` JSONB
column on `user_agents`.

No new pages, routes, or agent dashboards are introduced in this phase. Generative UI
chat interfaces are Phase 17. This spec governs the cadence config UI surface only.

## Interaction Contracts

### Toggle Row Interaction
- Each Switch row is independently controllable.
- Changing any toggle triggers immediate debounced save (500ms) — no explicit "Save" button.
- While `isSaving`, show `text-xs text-muted-foreground` "Saving..." below the last row (same as HeartbeatConfigSection).
- If save fails, show sonner toast: "Failed to save cadence settings. Try again." (destructive variant).

### Disabled State
- When the agent's `heartbeat_enabled = false` (global heartbeat off), render the entire cadence tiers group with `opacity-50 pointer-events-none` and add caption below trigger: "Enable heartbeat checks above to configure cadence."
- Event triggers remain independently toggleable even if all cadence tiers are disabled (they can fire without scheduled runs).

### CadenceConfigSection placement
- Insert `<CadenceConfigSection agentTypeId={agentTypeId} />` directly below `<HeartbeatConfigSection>` in `GenericAgentPanel`, before the closing panel content area.
- Same `border-t` top border as HeartbeatConfigSection for visual separation.

## States to Implement

| State | Trigger | Visual |
|-------|---------|--------|
| Loading | cadence_config fetch in flight | 5x Skeleton rows at `h-8 w-full` (matches HeartbeatConfigSection pattern) |
| Loaded default | cadence_config returned | Switch rows with values from DB (daily=true, weekly=true, monthly=false, quarterly=false, event=true) |
| Saving | any toggle changed within 500ms debounce window | "Saving..." text-xs muted below last row |
| Save error | Supabase upsert throws | Sonner toast "Failed to save cadence settings. Try again." destructive variant |
| Heartbeat disabled | heartbeat_enabled = false on parent config | Cadence tiers group: opacity-50 pointer-events-none, helper caption rendered |
| Collapsible closed | default (isOpen=false) | Only trigger row visible |
| Collapsible open | user clicks trigger | Full config form revealed with accordion animation |
