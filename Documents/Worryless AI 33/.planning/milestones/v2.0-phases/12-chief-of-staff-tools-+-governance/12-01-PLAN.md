    **File 2: `governance/token-budget.ts`**

    Follow the pg.Pool singleton pattern. Export:

    ```typescript
    export interface BudgetStatus {
      paused: boolean;       // tokens_used >= budget AND no active override
      warned: boolean;       // tokens_used >= 80% of budget
      usedPct: number;       // percentage of budget consumed (0-100+)
      remaining: number;     // tokens remaining (can be negative)
    }

    export async function checkTokenBudget(userId: string, agentTypeId: string): Promise<BudgetStatus>
    export async function incrementTokenUsage(userId: string, agentTypeId: string, tokens: number): Promise<void>
    export async function resetMonthlyBudget(userId: string, agentTypeId: string): Promise<void>
    ```

    `checkTokenBudget`:
    1. SELECT monthly_token_budget, tokens_used_this_month, budget_reset_at, budget_override_until FROM public.user_agents WHERE user_id = $1 AND agent_type_id = $2
    2. If no row found, return { paused: false, warned: false, usedPct: 0, remaining: 100000 }
    3. If budget_reset_at < NOW(), call resetMonthlyBudget and return fresh status
    4. Calculate pct = (tokens_used / budget) * 100
    5. Check budget_override_until for active override
    6. Return { paused: pct >= 100 && !hasOverride, warned: pct >= 80, usedPct: pct, remaining: budget - tokens_used }

    `incrementTokenUsage`:
    ```sql
    UPDATE public.user_agents
    SET tokens_used_this_month = tokens_used_this_month + $3
    WHERE user_id = $1 AND agent_type_id = $2
    ```

    `resetMonthlyBudget`:
    ```sql
    UPDATE public.user_agents
    SET tokens_used_this_month = 0,
        budget_reset_at = date_trunc('month', NOW() + INTERVAL '1 month')
    WHERE user_id = $1 AND agent_type_id = $2
    ```

    **File 3: `governance/task-checkout.ts`**

    Follow the pg.Pool singleton pattern. Export:

    ```typescript
    export async function atomicCheckoutTask(taskId: string, claimedBy: string): Promise<boolean>
    export async function releaseTask(taskId: string, newStatus: "completed" | "failed"): Promise<void>
    ```

    `atomicCheckoutTask`:
    ```sql
    UPDATE public.agent_tasks
    SET claimed_by = $2, claimed_at = NOW(), status = 'running'
    WHERE id = $1 AND claimed_by IS NULL AND status IN ('pending', 'scheduled')
    RETURNING id
    ```
    Returns `result.rowCount === 1`. This is the core atomic checkout — no advisory locks needed because UPDATE WHERE ... RETURNING is atomic under READ COMMITTED isolation.

    `releaseTask`:
    ```sql
    UPDATE public.agent_tasks
    SET claimed_by = NULL, claimed_at = NULL, status = $2, last_run_at = NOW()
    WHERE id = $1
    ```
