<objective>
Add agent tool boundary enforcement to the orchestrator edge function. When the orchestrator delegates a task to a specific agent, it must include that agent's skill_config in the system prompt so the agent knows which tool categories it can use — and explicitly cannot use tools outside that list.

Purpose: TOOLS-04 requires that an HR agent cannot trigger invoice functions and a Sales agent cannot trigger calendar writes. Without this guard, any agent can theoretically call any tool through the orchestrator, which defeats the role-based tooling model.
Output: Modified orchestrator that injects TOOL BOUNDARIES section into delegated agent prompts; human verification that routing behavior is correct.

**The approach:** Make `buildAgentPrompt` async and have it fetch the agent's `skill_config` from `available_agent_types`, then append a TOOL BOUNDARIES section to the system prompt. Because `buildAgentPrompt` is called from multiple places (lines 492, 529, 616), the change must be backward compatible.

**Change 1: Make buildAgentPrompt async and accept supabaseAdmin:**

```typescript
async function buildAgentPrompt(
  agentKey: string,
  businessKnowledge: string,
  supabaseAdmin?: ReturnType<typeof createClient>
): Promise<string> {
  const agent = baseAgentPrompts[agentKey as keyof typeof baseAgentPrompts];
  if (!agent) return "";

  let toolBoundarySection = "";
  if (supabaseAdmin) {
    try {
      const { data: agentType } = await supabaseAdmin
        .from('available_agent_types')
        .select('skill_config, display_name')
        .eq('id', agentKey)
        .single();

      if (agentType?.skill_config && Array.isArray(agentType.skill_config) && agentType.skill_config.length > 0) {
        toolBoundarySection = `

TOOL BOUNDARIES: You are the ${agentType.display_name || agentKey}. You are ONLY permitted to use tools and capabilities in this category list: ${agentType.skill_config.join(', ')}.
Do not attempt actions outside your designated tool categories. Stay within your role boundaries.`;
      }
    } catch {
      // Non-blocking — if DB lookup fails, prompt still works without boundary section
    }
  }

  return agent.basePrompt + businessKnowledge + toolBoundarySection;
}
```
