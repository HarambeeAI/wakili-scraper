`trackBudgetVsActual(userId: string): Promise<BudgetComparison[]>`:
- Read budget targets from LangGraph Store: `getStore(\`${userId}:agent_memory:accountant\`, "budget_targets")`
- If no budget targets found, return empty array (user hasn't set budgets yet)
- Parse stored value as `BudgetTarget[]`
- Query actual spending by category for current month:
  ```sql
  SELECT COALESCE(category, 'uncategorized') as category, SUM(amount) as total
  FROM public.transactions
  WHERE user_id = $1 AND type = 'expense'
    AND date >= date_trunc('month', CURRENT_DATE)
  GROUP BY 1
  ```
- For each budget target, find matching actual spending and compute: variance = budgeted - actual, variancePct = (actual / budgeted) * 100
- Return BudgetComparison[]

**2. Create `src/tools/accountant/anomaly-tools.ts` (ACCT-09):**

Import `getPool` from `../shared/db.js`, types from `./types.js`.

`detectAnomalousTransactions(userId: string): Promise<AnomalousTransaction[]>`:
- Query all transactions from last 90 days:
  ```sql
  SELECT id, amount, description, date::text, COALESCE(category, 'uncategorized') as category
  FROM public.transactions
  WHERE user_id = $1 AND date >= NOW() - INTERVAL '90 days'
  ORDER BY date DESC
  ```
- Group by category, compute stats per category:
  - mean = sum(amounts) / count
  - stdDev = sqrt(sum((amount - mean)^2) / count)
- For each transaction, calculate z-score = |amount - mean| / stdDev (where stdDev > 0)
- Flag transactions with z-score > 2.0 as anomalous
- For each flagged transaction, set reason string: "Amount ${amount} is ${zScore.toFixed(1)} standard deviations from the ${category} average of ${mean.toFixed(2)}"
- Return top 20 anomalous transactions sorted by z-score descending

`forecastRunway(userId: string): Promise<RunwayForecast>`:
- Query cash balance:
  ```sql
  SELECT COALESCE(SUM(CASE WHEN type='income' THEN amount ELSE -amount END), 0) as cash_balance
  FROM public.transactions WHERE user_id = $1
  ```
- Query monthly averages (last 3 months):
  ```sql
  SELECT type, AVG(monthly_total) as avg_monthly
  FROM (
    SELECT type, date_trunc('month', date) as month, SUM(amount) as monthly_total
    FROM public.transactions
    WHERE user_id = $1 AND date >= NOW() - INTERVAL '3 months'
    GROUP BY type, date_trunc('month', date)
  ) sub
  GROUP BY type
  ```
- Calculate: monthlyBurnRate = avg expense, monthlyIncome = avg income, netBurn = monthlyBurnRate - monthlyIncome
- runwayMonths = netBurn > 0 ? cashBalance / netBurn : 999 (infinite runway)
- status: runwayMonths >= 6 ? "healthy" : runwayMonths >= 2 ? "warning" : "critical"
- Return RunwayForecast

`generateInvoiceHtml(userId: string, invoiceId: string): Promise<string>`:
- Query invoice + related transactions:
  ```sql
  SELECT i.*, json_agg(t.*) as line_items
  FROM public.invoices i
  LEFT JOIN public.transactions t ON t.invoice_id = i.id
  WHERE i.id = $1 AND i.user_id = $2
  GROUP BY i.id
  ```
- Call callLLM to generate structured HTML invoice from the data, systemPrompt: "Generate a clean, professional HTML invoice. Use inline CSS. Include: company header, invoice number, date, vendor details, line items table, totals, payment terms."
- Store in agent_assets:

  ```sql
  INSERT INTO public.agent_assets (user_id, agent_type, asset_type, title, content, metadata)
  VALUES ($1, 'accountant', 'document', $2, $3, $4)
  RETURNING id
  ```
  Where title = "Invoice ${invoiceId}", content = HTML string, metadata = { invoiceId, generatedAt: new Date().toISOString() }
- Return the asset id
