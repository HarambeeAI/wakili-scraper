`detectStaleDeals(userId: string): Promise<StaleDeal[]>`:
- Query leads not in terminal states with staleness:
  ```sql
  SELECT id, company_name, contact_name, status, score, deal_value, updated_at
  FROM public.leads
  WHERE user_id = $1
    AND status NOT IN ('converted', 'closed_won', 'closed_lost', 'lost')
    AND updated_at < NOW() - INTERVAL '5 days'
  ORDER BY updated_at ASC
  LIMIT 20
  ```

**3. Create `src/tools/sales/pipeline-tools.ts` (SALES-10, SALES-11):**

Import `getPool` from `../shared/db.js`, types from `./types.js`.

`analyzePipeline(userId: string): Promise<PipelineAnalysis>`:
- Query deal counts and value by stage:
  ```sql
  SELECT
    status,
    COUNT(*)::int as deal_count,
    COALESCE(SUM(score), 0)::int as total_score,
    SUM(deal_value) as total_deal_value,
    AVG(EXTRACT(epoch FROM (NOW() - updated_at)) / 86400)::numeric(10,1) as avg_days_in_stage
  FROM public.leads
  WHERE user_id = $1
  GROUP BY status
  ```
- Query historical conversion rate (last 90 days):
  ```sql
  SELECT
    COUNT(*) FILTER (WHERE status IN ('converted', 'closed_won'))::int as converted,
    COUNT(*)::int as total
  FROM public.leads
  WHERE user_id = $1
    AND created_at >= NOW() - INTERVAL '90 days'
  ```
- Calculate totals: totalDeals = sum of all deal_count, totalValue = sum of all total_deal_value
- conversionRate = total > 0 ? converted / total : 0
- Return PipelineAnalysis

`forecastRevenue(userId: string): Promise<RevenueForcast>`:
- Get pipeline data from analyzePipeline call
- Define stage conversion weights:
  ```typescript
  const STAGE_WEIGHTS: Record<string, number> = {
    prospecting: 0.05, contacted: 0.10, responded: 0.20,
    qualified: 0.40, proposal: 0.60, converted: 1.0, closed_won: 1.0,
  };
  ```
- Calculate weighted pipeline: for each stage, sum(deal_value * weight)
- Get historical monthly revenue (last 3 months of closed deals):
  ```sql
  SELECT AVG(monthly_revenue) as avg_monthly FROM (
    SELECT date_trunc('month', updated_at) as month, SUM(deal_value) as monthly_revenue
    FROM public.leads
    WHERE user_id = $1
      AND status IN ('converted', 'closed_won')
      AND updated_at >= NOW() - INTERVAL '3 months'
    GROUP BY 1
  ) sub
  ```
- projected30d = weightedPipeline * 0.33 + avgMonthly
- projected60d = weightedPipeline * 0.55 + avgMonthly * 2
- projected90d = weightedPipeline * 0.80 + avgMonthly * 3
- Return RevenueForcast
